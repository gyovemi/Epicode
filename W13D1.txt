TASK 1

Fare la spesa

1 GO TO Dispensa

2 CHECK AVAILABILITY of Dispensa = [Latte, Pasta, Acqua]

3 CREATE List

4 IF Dispensa == Null 
OR Latte == 0
OR Pasta == 0
OR Acqua == 0 :
	GO TO Mercato AND BUY in Mercato Dispensa
ELSE STAY Home

5 GO BACK Home

TASK 2

Studiare un concetto

1 SELECT MATERIA = Lista ARGOMENTO 

2 SELECT ARGOMENTO = Lista CONCETTO

3 READ ARGOMENTO

4 ID CONCETTO

5 STUDY CONCETTO

6 SYNTETIZE CONCETTO

7 IF CONCETTO IS UNDERSTOOD:
	GO TO NEXT CONCETTO
ELSE REPEAT FROM STEP 5

TASK 3

Riempire il serbatio dell'Auto

1 GO TO Macchina

2 Check Indicatore Carburante

LOOP

3 IF Indicatore < 50%:
	GO TO Distributore di Carburante AND FILL Carburante
ELSE Do Nothing

4 IF Indicatore == 100%
	EXIT LOOP
ELSE REPEAT FROM STEP 3
